Example Student's HCDE 439 Physical Computing Page!

My name is Sam Ding! I'm an HCDE major and excited to learn the following things in this class: how to do physical computing and explore my interest in this class.

Assignment 1: Blink!

Assignment 1 was to create a schematic for at least 3 LEDs connected to your Arduino and to write firmware (in a .ino file) for the circuit that makes the LEDs blink in a pattern.

Assignment 2: Fade!

Assignment 2 was to create a schematic, circuit, and code that uses LEDs, a button, a for-loop, digitalWrite(), digitalRead(), and analogWrite().

Assignment 3: Input Output!

Assignment 3 was to create a schematic, circuit, and code that uses a sensor (e.g. either the photoresistor or the thermistor) in a voltage divider to change the state of an LED.

Assignment 4: High(er) voltage and transistors!

Assignment 4 was to create a schematic, circuit, and code that uses a transistor to control load power separate from logic power.
